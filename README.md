# ChuckleChick

An extensible Windows desktop pet and assistant powered by the laughing yellow chick.

## Current contents

The repository currently contains the approved **黄鸡大笑** character assets:

- `assets/pets/huangji-daxiao/spritesheet.png` — transparent 8 × 11 animation atlas.
- `assets/pets/huangji-daxiao/pet.json` — pet identity and asset entry point.
- `assets/pets/huangji-daxiao/atlas.json` — cell geometry, animation rows, timing, and look directions.

No desktop application code has been added yet.

## Sprite atlas

- Canvas: `1536 × 2288`
- Grid: `8 columns × 11 rows`
- Cell: `192 × 208`
- Format: transparent PNG
- Rows 0–8: standard animation states
- Rows 9–10: 16 clockwise look directions
